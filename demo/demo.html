jquery.popups.js

A css driven jquery plug-in for all your popup and dialog needs

About

Getting Started

$('<div class="my-class">').popup();

Will simply Render:

<div class="popup-container">
	<div class="my-class"></div>
</div>

An example where you already have the markup in the DOM

$('.notification').popup();

Will turn this:

...
<aside>
	<div class="notification">This is a notification!</div>
</aside>
...
This is a notification!

Into this:

...
<aside>
	<div class="popup-container">
		<div class="notification">This is a notification!</div>
	</div>
</aside>
...
This is a notification!

A full example with several options passed

$('.notification').popup({
	attachTo: '.notify-btn',
	classes: 'bubble',
	collision: false,
	position: right,
	showArrow: true,
	showClose: true
});

Will turn this:

...
<aside>
	<button class="btn notify-btn">Notify!</button>
	<div class="notification">This is a notification!</div>
</aside>
...
This is a notification!

Into this:

...
<aside>
	<button class="btn notify-btn">Notify!</button>
	<div class="popup-container">
		<button class="popup-close" type="button"></button>
		<div class="popup-arrow">
			<div class="inner-arrow"></div>
		</div>
		<div class="notification">This is a notification!</div>
	</div>
</aside>
...
This is a notification!

API

Options

The object properties that can be passed on creation of a popup

Name Type Default Description
attachTo jQuery | element | string null
autoShow boolean true
boundary number 10
classes string null
collision string 'fitflip'
container jQuery | element | string null
destroyOnHide boolean false
offset string '50%'
placement string 'right'
showArrow boolean true
showClose boolean true

Methods

There are methods that can be called for your programming convenience.

Name Description
show
hide
toggle
destroy
setContent

Properties

Get only properties for your programming convenience

Name Return Type Description
$popup jQuery
$el jQuery
isShowing boolean

Events

This are the events you can hook into. All the Events are called on the element .popup() was called upon.

Name Description
create.popup
show.popup
shown.popup
positioned.popup
hide.popup
hidden.popup
destroy.popup

jqXHR

$('#jqXHR-test').on('click', function() {

	var echo = $('#jqXHR-echo').val();

	var $ajax = $.ajax({
		type: 'GET',
		url: 'http://jsfiddle.net/echo/jsonp/',
		data: {
			html: '
Here is your Echo:
',
			echo: echo,
			delay: 3
		},
		dataType: 'jsonp'
	}).done(function(data) {
		$echoPopup.popup('setContent', data.html);
		$echoPopup.find('.echo').text(data.echo);
	});

	var $echoPopup = $.popup($ajax, {
		attachTo: '#jqXHR-test',
		classes: 'bubble',
		collision: false,
		placement: 'right',
		destroyOnHide: true
	});
});
