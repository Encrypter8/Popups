jquery.popups.js

A css driven jquery plug-in for all your popup and dialog needs

About

Getting Started

$('<div class="my-class">').popup();

Will simply Render:

<div class="popup-container">
	<div class="my-class"></div>
</div>

An example where you already have the markup in the DOM

$('.notification').popup();

Will turn this:

...
<aside>
	<div class="notification">This is a notification!</div>
</aside>
...
This is a notification!

Into this:

...
<aside>
	<div class="popup-container">
		<div class="notification">This is a notification!</div>
	</div>
</aside>
...
This is a notification!

A full example with several options passed

$('.notification').popup({
	attachTo: '.notify-btn',
	classes: 'bubble',
	collision: false,
	position: right,
	showArrow: true,
	showClose: true
});

Will turn this:

...
<aside>
	<button class="btn notify-btn">Notify!</button>
	<div class="notification">This is a notification!</div>
</aside>
...
This is a notification!

Into this:

...
<aside>
	<button class="btn notify-btn">Notify!</button>
	<div class="popup-container">
		<button class="popup-close" type="button"></button>
		<div class="popup-arrow">
			<div class="inner-arrow"></div>
		</div>
		<div class="notification">This is a notification!</div>
	</div>
</aside>
...
This is a notification!
